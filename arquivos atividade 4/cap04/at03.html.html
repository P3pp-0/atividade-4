Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem utilizada para estilizar elementos HTML.
JavaScript
Linguagem de programação usada para adicionar interatividade às páginas.
DOM
Representação estruturada dos elementos de uma página HTML.
Interface que permite manipular conteúdos e eventos via JavaScript.
API
Conjunto de regras e recursos para comunicação entre sistemas.
Interface usada para integração de aplicações e serviços.
HTTP
HyperText Transfer Protocol
Protocolo utilizado para transferência de dados na web.
URL
Endereço Web
Localização de um recurso disponível na internet.
DNS
Sistema responsável por converter nomes de domínio em endereços IP.
